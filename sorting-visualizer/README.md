# Sort Studio

Animated sorting algorithm visualizer (Bubble / Selection / Insertion / Quick / Merge / Heap) with:

- Generate random / nearly-sorted / reversed datasets (seeded)
- Start / Pause / Resume / Step / Reset
- "Show all algorithms" mode (same dataset + synced playback)
- Speed control and live metrics (comparisons, swaps, writes, accesses, elapsed)
- Reduced-motion support via OS setting

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
